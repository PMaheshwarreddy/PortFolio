# Panduga Maheswar Reddy Portfolio - MERN

Futuristic MERN portfolio converted from `portfolio-v3.html`.

## Structure

- `frontend` - Vite + React portfolio UI
- `backend` - Express API for profile and contact form
- `frontend/public/assets/profile-cutout-ai.png` - enhanced transparent profile asset
- `frontend/public/assets/profile-original.jpeg` - original profile photo copy

## Run Locally

```bash
npm install
npm run dev
```

Frontend: `http://localhost:5173`  
Backend: `http://localhost:5000`

Copy `frontend/.env.example` and `backend/.env.example` to `.env` files when you want custom API URL, MongoDB persistence, or SMTP email sending.
